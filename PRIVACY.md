# Privacy Policy

Copy Clean URL does not collect, store, or transmit any personal data or browsing information.

The extension reads the URL of the current tab solely to remove tracking parameters and copy the cleaned URL to your clipboard. This processing happens entirely within your browser. No data is sent to any external server.

## Permissions

- `tabs`: Used to read the current tab's URL for cleaning.
- `clipboardWrite`: Used to write the cleaned URL to your clipboard.
- `scripting`: Used to display a confirmation toast notification on the current page.
- `activeTab`: Used to access the currently active tab when the shortcut is triggered.
